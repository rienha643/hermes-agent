비추천 이유: 아카라이브 사용자 경험상 스타일 변화 체감이 있고, 고가중치에서 손만 과도하게 강조될 수 있습니다.

OrangeMix 사용 사례: 직접 OrangeMix 표기는 찾지 못했지만, SD1.5 애니 계열용으로 설계되었고 Anything 계열/애니 모델 사용 사례가 확인됩니다.

NAI 사용 사례: 직접 NAI 사례는 부족합니다. NAI 감성 강화용보다는 로컬 SD1.5 손 보정용에 가깝습니다.

브라운더스트2 적합성: 브라운더스트2 취향에서 손·포즈 안정성 확보용 보조 LoRA로는 유효하나, 그림체 보정 LoRA는 아닙니다.

데스티니 차일드 적합성: 데스티니 차일드 계열에서도 손 노출 컷 안정화에는 도움될 수 있으나, 스타일 적합성 근거는 약합니다.

장점

2. Front face lighting

다운로드 3,552 좋아요 390 싫어요 0 공개 예시 이미지 7장

판정

추천 이유: 얼굴 그림자 제거/정면 보광이라는 목적이 분명하고, 커뮤니티 반응과 공개 사용자 이미지가 많습니다.

비추천 이유: 제작자도 광학적으로 부자연스러울 수 있다고 경고하며, 과가중치 시 과조명/도장 뭉개짐이 발생할 수 있습니다.

OrangeMix 사용 사례: 직접 OrangeMix 표기는 부족하지만 SD1.5 애니/일러스트 계열 사용이 폭넓게 확인됩니다.

NAI 사용 사례: NAI 직접 사례보다는 로컬 SD1.5 얼굴 보정 LoRA에 가깝습니다.

브라운더스트2 적합성: 브라운더스트2 취향의 “얼굴이 어둡지 않고 전면이 화사한” 컷과 잘 맞습니다.

데스티니 차일드 적합성: 데스티니 차일드풍에서도 얼굴 강조 컷에는 적합하지만, 전신/역광 연출에서는 과개입 위험이 있습니다.

장점 / 단점

장점

  • 다운로드/평가량이 높음
  • 실제 사용자 이미지가 풍부함
  • Detail Tweaker와 병행 사례가 확인됨

단점

  • 부자연스러운 광학 효과 가능
  • 세부 묘사 약화 시 보조 LoRA 필요

실사용 사례 요약

  • Civitai 공개 이미지 샘플에서 DreamShaper + Front face lighting + 스타일/디테일 LoRA 병행 사례가 확인됩니다.
  • 상위 샘플에서 Detail Tweaker 병행 빈도가 눈에 띄며, 실제 운영에서 얼굴 보광과 세부 보정이 함께 쓰이는 패턴이 보입니다.
  • 아카라이브에서는 얼굴 그림자 문제가 반복적으로 제기되어, 이 LoRA의 해결 목표와 수요가 직접적으로 맞물립니다.

3. Maidoll Style

다운로드 5,525 좋아요 924 싫어요 2 공개 예시 이미지 11장

판정

추천 이유: OrangeMix·NovelAI VAE 계열 호환성을 제작자가 직접 명시했고, NAI 감성 강화 후보 중 근거가 가장 분명합니다.

비추천 이유: 이번에 확인한 공개 사용자 이미지들은 범용 스타일 실험 비중이 높아 BD2/DC 취향 직결 사례는 부족합니다.

OrangeMix 사용 사례: 모델 설명에서 Orangemix를 권장 784MB VAE 계열로 직접 언급합니다.

NAI 사용 사례: NovelAI VAE 또는 kl-f8-anime2 VAE 권장이 명시되어 있습니다.

브라운더스트2 적합성: 고광택 미소녀 일러스트 톤에는 일부 맞지만, BD2 특유의 의상/체형 연출 직결 사례는 부족합니다.

데스티니 차일드 적합성: 화사한 미소녀 일러스트 방향성은 맞아도 Kim Hyung-Tae 계열만큼 직접적이지는 않습니다.

장점 / 단점

장점

  • OrangeMix/NAI 계열 호환성 근거가 명시적
  • 커뮤니티 평가량이 높음
  • 프리뷰 이미지 수가 많음

단점

  • 실사용 이미지가 스타일 혼합 위주
  • BD2/DC 직결 사례가 적음

실사용 사례 요약

  • 제작자 설명에서 non-NAI 체크포인트 호환 수정, NovelAI/Orangemix 권장 VAE 명시가 확인됩니다.
  • Civitai 공개 사용자 이미지에서는 maidoll 트리거를 직접 넣어 스타일을 호출하는 사례가 확인됩니다.
  • 이번 샘플에서는 “실제 사용은 존재하나, BD2/DC 지향으로 수렴하는지”는 추가 검증이 필요했습니다.

장점
